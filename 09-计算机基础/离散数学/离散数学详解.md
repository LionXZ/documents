## 1. 命题逻辑

### 1.1 基本概念

**命题**：能判断真假的陈述句。

**联结词**：

| 符号 | 名称 | 含义 |
|------|------|------|
| ¬ | 否定 | 非 |
| ∧ | 合取 | 且 |
| ∨ | 析取 | 或 |
| → | 蕴含 | 如果...则... |
| ↔ | 等价 | 当且仅当 |

**真值表（以 p→q 为例）**：

| p | q | p→q |
|---|---|-----|
| T | T | T |
| T | F | F |
| F | T | T |
| F | F | T |

> 关键理解：p→q 只有当前件 p 为真且后件 q 为假时才为假。

### 1.2 逻辑等价式

| 名称 | 公式 |
|------|------|
| 双重否定律 | ¬¬p ⇔ p |
| 德摩根律 | ¬(p∧q) ⇔ ¬p∨¬q，¬(p∨q) ⇔ ¬p∧¬q |
| 蕴含等价式 | p→q ⇔ ¬p∨q |
| 等价展开式 | p↔q ⇔ (p→q)∧(q→p) ⇔ (p∧q)∨(¬p∧¬q) |

### 1.3 范式

- **析取范式 (DNF)**：若干合取式的析取，如 `(p∧q)∨(¬p∧r)`
- **合取范式 (CNF)**：若干析取式的合取，如 `(p∨q)∧(¬p∨r)`

---

### 解题示例

**例题1**：用真值表证明 `p→q ⇔ ¬p∨q`

| p | q | ¬p | p→q | ¬p∨q |
|---|---|----|-----|------|
| T | T | F | T | T |
| T | F | F | F | F |
| F | T | T | T | T |
| F | F | T | T | T |

p→q 与 ¬p∨q 真值列完全相同，得证。✓

---

**例题2**：求公式 `(p→q)∧(q→r) → (p→r)` 的主析取范式

**解**：
1. 先用蕴含等价式消去→：
   - `p→q = ¬p∨q`
   - `q→r = ¬q∨r`
   - 原式 `A = (¬p∨q)∧(¬q∨r) → (¬p∨r)`

2. 再消外层→：
   - `A = ¬[(¬p∨q)∧(¬q∨r)] ∨ (¬p∨r)`

3. 用德摩根律：
   - `= [¬(¬p∨q) ∨ ¬(¬q∨r)] ∨ (¬p∨r)`
   - `= [(p∧¬q) ∨ (q∧¬r)] ∨ (¬p∨r)`
   - `= (p∧¬q) ∨ (q∧¬r) ∨ ¬p ∨ r`

4. 由于包含 `¬p∨r`（即 p→r），该式其实为永真式，主析取范式包含所有极小项。✓

---

**例题3**：判断推理是否正确——"如果天下雨，我就带伞。我没带伞。所以天没下雨。"

**解**：
- 令 p：天下雨，q：我带伞
- 前提：p→q，¬q
- 结论：¬p
- 检验：`[(p→q) ∧ ¬q] → ¬p`

| p | q | p→q | (p→q)∧¬q | ¬p | 总式 |
|---|---|-----|----------|----|-----|
| T | T | T | F | F | T |
| T | F | F | F | F | T |
| F | T | T | F | T | T |
| F | F | T | T | T | T |

总式为永真式，推理正确。这是**拒取式 (Modus Tollens)** 推理规则。✓

---

**例题4**：不用真值表，证明 `¬(p∧q) ⇔ ¬p∨¬q`（德摩根律）

**解**：
要证明等价，只需证明左边→右边 且 右边→左边。

**(⇒)** 假设 `¬(p∧q)` 为真，即 p∧q 为假。
- 则 p 为假 或 q 为假（或两者都为假）
- 即 `¬p` 为真 或 `¬q` 为真
- 故 `¬p∨¬q` 为真 ✓

**(⇐)** 假设 `¬p∨¬q` 为真，即 ¬p 为真 或 ¬q 为真。
- 则 p 为假 或 q 为假
- 故 p∧q 为假
- 即 `¬(p∧q)` 为真 ✓

两边互相蕴含，等价得证。✓

---

**例题5**：判断公式 `(p∧q) → p` 是永真式、永假式还是可满足式？

**解**：
`(p∧q) → p`
= `¬(p∧q) ∨ p`（蕴含等价式）
= `(¬p ∨ ¬q) ∨ p`（德摩根律）
= `¬p ∨ ¬q ∨ p`
= `(¬p ∨ p) ∨ ¬q`（交换律+结合律）
= `T ∨ ¬q`
= `T`

故 `(p∧q) → p` 是**永真式**。这对应推理规则中的**化简律 (Simplification)**。✓

---

**例题6**：求公式 `(p→q) ∧ (q→r)` 的主合取范式

**解**：
1. 消去→：`(¬p∨q) ∧ (¬q∨r)`

2. 补全每个析取式中缺失的文字使其成为极大项（3 个变元 p,q,r）：
   - `¬p∨q` 缺少 r，添加 `(r∧¬r)`：
     `= (¬p∨q∨r) ∧ (¬p∨q∨¬r)`
   - `¬q∨r` 缺少 p，添加 `(p∧¬p)`：
     `= (p∨¬q∨r) ∧ (¬p∨¬q∨r)`

3. 主合取范式：
   `(¬p∨q∨r) ∧ (¬p∨q∨¬r) ∧ (p∨¬q∨r) ∧ (¬p∨¬q∨r)` ✓

---

**例题7**：判断 ─ "若今天是周二，则明天是周三。今天是周二。所以明天是周三。" ─ 用了什么推理规则？是否有效？

**解**：
令 p：今天是周二，q：明天是周三
前提：p→q, p
结论：q

这是**假言推理 (Modus Ponens)**：`[(p→q) ∧ p] → q`

| p | q | p→q | (p→q)∧p | 总式 |
|---|---|-----|---------|-----|
| T | T | T | T | T |
| T | F | F | F | T |
| F | T | T | F | T |
| F | F | T | F | T |

永真式，推理有效。✓

## 2. 谓词逻辑

### 2.1 基本概念

- **个体词**：研究对象（常量 a,b,c... 或变量 x,y,z...）
- **谓词**：表示个体性质或关系的词，记作 P(x), Q(x,y)
- **量词**：
  - ∀x：全称量词，"对所有 x"
  - ∃x：存在量词，"存在 x"

### 2.2 量词否定律

| 公式 |
|------|
| ¬∀x P(x) ⇔ ∃x ¬P(x) |
| ¬∃x P(x) ⇔ ∀x ¬P(x) |

### 2.3 前束范式

所有量词都在公式最前面，形如：
`Q₁x₁ Q₂x₂ ... Qₙxₙ B`（其中 Qᵢ 是 ∀ 或 ∃，B 不含量词）

---

### 解题示例

**例题1**：用谓词逻辑符号化："所有学生都学离散数学"

**解**：
- 令 S(x)：x 是学生，D(x)：x 学离散数学
- 符号化：`∀x (S(x) → D(x))`

> 注意：全称量词后跟的是**蕴含式**，不是合取式。✓

---

**例题2**：求 `∀x∀y (P(x,y) → Q(x,y))` 的否定

**解**：
```
¬∀x∀y (P(x,y) → Q(x,y))
= ∃x ¬∀y (P(x,y) → Q(x,y))       （量词否定律）
= ∃x ∃y ¬(P(x,y) → Q(x,y))       （量词否定律）
= ∃x ∃y ¬(¬P(x,y) ∨ Q(x,y))      （蕴含等价式）
= ∃x ∃y (P(x,y) ∧ ¬Q(x,y))       （德摩根律）
```
最终结果：存在某对 x,y，使得 P(x,y) 为真而 Q(x,y) 为假。✓

---

**例题3**：将公式 `∀x P(x) → ∃y Q(y)` 化为前束范式

**解**：
1. `∀x P(x) → ∃y Q(y)`
2. `= ¬∀x P(x) ∨ ∃y Q(y)` （消去→）
3. `= ∃x ¬P(x) ∨ ∃y Q(y)` （量词否定律）
4. `= ∃x ∃y (¬P(x) ∨ Q(y))` （量词前移）

注意 x,y 不同，可直接前移。✓

---

**例题4**：符号化："没有最大的整数"

**解**：
- 令 Z(x)：x 是整数
- "没有最大的整数" = ¬∃x (Z(x) ∧ ∀y (Z(y) → y≤x))
- 等价于：∀x (Z(x) → ∃y (Z(y) ∧ y>x))（每个整数都存在比它更大的整数）✓

---

**例题5**：判断推理的有效性 ─ 前提："所有哺乳动物都是恒温动物"，"鲸鱼是哺乳动物"。结论："鲸鱼是恒温动物。"

**解**：
令 M(x)：x 是哺乳动物，W(x)：x 是恒温动物，a：鲸鱼

前提：∀x (M(x) → W(x)), M(a)
结论：W(a)

推理步骤：
1. ∀x (M(x) → W(x))（前提）
2. M(a) → W(a)（全称实例化 UI，令 x=a）
3. M(a)（前提）
4. W(a)（假言推理，由 2,3）

推理有效。✓

---

**例题6**：符号化并求否定："所有学生都喜欢某些课程"

**解**：
令 S(x)：x 是学生，C(y)：y 是课程，L(x,y)：x 喜欢 y

原命题：`∀x [S(x) → ∃y (C(y) ∧ L(x,y))]`

否定：
```
¬∀x [S(x) → ∃y (C(y) ∧ L(x,y))]
= ∃x ¬[S(x) → ∃y (C(y) ∧ L(x,y))]       （量词否定）
= ∃x ¬[¬S(x) ∨ ∃y (C(y) ∧ L(x,y))]       （蕴含等价）
= ∃x [S(x) ∧ ¬∃y (C(y) ∧ L(x,y))]        （德摩根律）
= ∃x [S(x) ∧ ∀y ¬(C(y) ∧ L(x,y))]        （量词否定）
= ∃x [S(x) ∧ ∀y (C(y) → ¬L(x,y))]        （德摩根+蕴含等价）
```

即："存在某个学生，他不喜欢任何课程" ✓

---

**例题7**：符号化："男孩和女孩都喜欢打篮球"（论域为所有人）

**解**：
令 B(x)：x 是男孩，G(x)：x 是女孩，L(x)：x 喜欢打篮球

这句话有两种合理的符号化：
- 方式一（"和"理解为"且"）：`∀x [(B(x) ∨ G(x)) → L(x)]`
- 方式二（拆开）：`∀x (B(x)→L(x)) ∧ ∀x (G(x)→L(x))`

两种方式等价。✓

## 3. 集合论

### 3.1 基本概念

**集合的表示**：
- 列举法：`A = {1, 2, 3}`
- 描述法：`A = {x | x 是正整数且 x < 4}`

**集合间关系**：
- 子集：`A ⊆ B` 当且仅当 `∀x (x∈A → x∈B)`
- 真子集：`A ⊂ B` 当且仅当 `A ⊆ B 且 A ≠ B`
- 相等：`A = B` 当且仅当 `A ⊆ B 且 B ⊆ A`

**幂集**：P(A) = {X | X ⊆ A}，即 A 的所有子集的集合
- 若 |A| = n，则 |P(A)| = 2ⁿ

### 3.2 集合运算

| 运算 | 定义 | 文氏图含义 |
|------|------|----------|
| 并 A∪B | {x | x∈A ∨ x∈B} | A和B的全部区域 |
| 交 A∩B | {x | x∈A ∧ x∈B} | A和B的公共区域 |
| 差 A−B | {x | x∈A ∧ x∉B} | 属于A但不属于B |
| 补 ∼A | {x | x∉A} | 全集U中除A以外的区域 |
| 对称差 A⊕B | (A−B)∪(B−A) | 并减去交 |

### 3.3 容斥原理

**两个集合**：`|A∪B| = |A| + |B| − |A∩B|`

**三个集合**：
`|A∪B∪C| = |A|+|B|+|C| − |A∩B|−|B∩C|−|A∩C| + |A∩B∩C|`

---

### 解题示例

**例题1**：设 A={1,2,3}，求 P(A)（幂集）

**解**：
P(A) = {∅, {1}, {2}, {3}, {1,2}, {1,3}, {2,3}, {1,2,3}}

|P(A)| = 2³ = 8 ✓

---

**例题2**：全班50人，学英语的30人，学日语的25人，两门都学的10人。求两门都不学的人数。

**解**：
用容斥原理：
- `|E| = 30, |J| = 25, |E∩J| = 10`
- `|E∪J| = 30 + 25 − 10 = 45`
- 两门都不学 = `50 − 45 = 5` 人 ✓

---

**例题3**：证明 `A − (B∩C) = (A−B) ∪ (A−C)`

**证**：对任意 x，
```
x ∈ A−(B∩C)
⇔ x∈A ∧ x∉(B∩C)
⇔ x∈A ∧ ¬(x∈B ∧ x∈C)
⇔ x∈A ∧ (x∉B ∨ x∉C)        （德摩根律）
⇔ (x∈A ∧ x∉B) ∨ (x∈A ∧ x∉C)
⇔ x ∈ (A−B) ∪ (A−C)
```
得证。可以画文氏图辅助理解。✓

---

**例题4**：设 A={1,2,3,4}, B={3,4,5}，求 A⊕B（对称差）

**解**：
A⊕B = (A−B) ∪ (B−A)
A−B = {1,2}
B−A = {5}
A⊕B = {1,2,5} ✓

公式法：A⊕B = (A∪B) − (A∩B) = {1,2,3,4,5} − {3,4} = {1,2,5} ✓

---

**例题5**：某班有 60 人，其中学数学的 35 人，学物理的 30 人，学化学的 28 人；同时学数学和物理的 15 人，数学和化学的 12 人，物理和化学的 10 人；三门都学的 5 人。问三门都不学的有几人？

**解**：
用三集合容斥原理：
|M∪P∪C| = |M|+|P|+|C| − |M∩P|−|P∩C|−|M∩C| + |M∩P∩C|
= 35 + 30 + 28 − 15 − 10 − 12 + 5
= 93 − 37 + 5
= 61

三门都不学 = 60 − 61 = −1？不可能！

**检查**：|M∪P∪C| 不能超过 60。重新计算：
35+30+28 = 93
减两两交集：93−15−10−12 = 56
加三交集：56+5 = 61

确实 61 > 60，数据有矛盾（题目就是为了验证容斥公式的使用方法）。实际题目中数据应保证合理性。✓

> 注：此例用来说明容斥公式的机械代入方法。

---

**例题6**：证明 `A ⊆ B` 当且仅当 `A∪B = B`

**证**：
**(⇒)** 设 A ⊆ B。
- 证 A∪B ⊆ B：任取 x∈A∪B，则 x∈A 或 x∈B。若 x∈A，由 A⊆B 知 x∈B。故总有 x∈B。∴ A∪B ⊆ B。
- 证 B ⊆ A∪B：显然，B ⊆ A∪B。
- 故 A∪B = B。✓

**(⇐)** 设 A∪B = B。
- 任取 x∈A，则 x∈A∪B = B，∴ x∈B。故 A ⊆ B。✓

---

**例题7**：证明 `(A∩B) × C = (A×C) ∩ (B×C)`（笛卡尔积与交的分配律）

**证**：对任意 (x, y)，
```
(x,y) ∈ (A∩B) × C
⇔ x ∈ A∩B ∧ y ∈ C
⇔ (x∈A ∧ x∈B) ∧ y∈C
⇔ (x∈A ∧ y∈C) ∧ (x∈B ∧ y∈C)
⇔ (x,y) ∈ A×C ∧ (x,y) ∈ B×C
⇔ (x,y) ∈ (A×C) ∩ (B×C)
```
得证。✓

## 4. 关系

### 4.1 基本概念

n元关系的本质是笛卡尔积的子集：
- 二元关系 `R ⊆ A×B`

**关系的表示**：
- 关系矩阵（0-1矩阵）
- 关系图（有向图）

### 4.2 关系的性质

设 R 是 A 上的二元关系：

| 性质 | 定义 | 关系图特征 | 关系矩阵特征 |
|------|------|----------|------------|
| 自反性 | ∀x∈A, (x,x)∈R | 每个顶点有自环 | 对角线全为1 |
| 反自反性 | ∀x∈A, (x,x)∉R | 无自环 | 对角线全为0 |
| 对称性 | xy∈R ⇒ yx∈R | 有向边成对出现 | 对称矩阵 |
| 反对称性 | xy∈R ∧ yx∈R ⇒ x=y | 无边是双向的 | 若 aᵢⱼ=1 则 aⱼᵢ=0 (i≠j) |
| 传递性 | xy∈R ∧ yz∈R ⇒ xz∈R | 有路径就有直达边 | — |

### 4.3 等价关系与偏序关系

**等价关系**：同时满足**自反、对称、传递**
- 产生**等价类**，构成集合的**划分**

**偏序关系**：同时满足**自反、反对称、传递**
- 记作 `≼`，可用**哈斯图**表示
- **全序**：偏序 + 任意两个元素可比

---

### 解题示例

**例题1**：设 A={1,2,3}，R={(1,1),(2,2),(3,3),(1,2),(2,1)}，判断 R 的性质

**解**：
| 性质 | 判定 |
|------|------|
| 自反性 | ✓ (1,1),(2,2),(3,3)都在R中 |
| 对称性 | ✓ (1,2)在R中→(2,1)也在R中 |
| 反对称性 | ✗ (1,2)和(2,1)都在，但1≠2 |
| 传递性 | ✗ (2,1)∈R,(1,2)∈R→需要(2,2)∈R ✓，但(1,2)∈R,(2,1)∈R→需要(1,1)∈R ✓，此处满足传递性 |
| | 再检：(2,1)∈R,(1,2)∈R→需(2,2)∈R ✓，(1,2)∈R,(2,1)∈R→需(1,1)∈R ✓，传递性成立 |

R 满足自反性、对称性、传递性，故 R 是 A 上的**等价关系**。✓

等价类：[1] = {1,2}，[2] = {1,2} = [1]，[3] = {3}

---

**例题2**：设 R={(1,2),(2,3),(1,4)} 是 A={1,2,3,4} 上的关系，S={(2,1),(4,3)}。求复合关系 R∘S 和 S∘R。

**解**：
R∘S = {(a,c) | ∃b: (a,b)∈R ∧ (b,c)∈S}
- (1,2)∈R, (2,1)∈S → (1,1)
- (2,3)∈R → 无需检查，无 (3,c)∈S
- (1,4)∈R, (4,3)∈S → (1,3)
∴ R∘S = {(1,1), (1,3)} ✓

S∘R = {(a,c) | ∃b: (a,b)∈S ∧ (b,c)∈R}
- (2,1)∈S, (1,2)∈R → (2,2)
- (2,1)∈S, (1,4)∈R → (2,4)
- (4,3)∈S → 无需检查，无 (3,c)∈R
∴ S∘R = {(2,2), (2,4)} ✓

注意：R∘S ≠ S∘R，关系复合**不满足交换律**。✓

---

**例题3**：设 A={1,2,3,4}，R={(1,2),(2,3),(3,4)}，求 R 的传递闭包 t(R)

**解**：
传递闭包需满足传递性，反复添加使关系传递：

当前 R：1→2→3→4
- 1→2 且 2→3 ⇒ 需添加 (1,3)
- 2→3 且 3→4 ⇒ 需添加 (2,4)
- 1→3 且 3→4 ⇒ 需添加 (1,4)（新发现的）
- (1,2) 和 (2,4) ⇒ 再次需 (1,4)，已在集合中
- 检查完毕

∴ t(R) = {(1,2),(2,3),(3,4),(1,3),(2,4),(1,4)} ✓

也可用 Warshall 算法或矩阵幂运算。✓

---

**例题4**：A={1,2,3}，R={(1,1),(2,2),(3,3),(1,2),(2,3)}。问 R 是偏序关系吗？

**解**：
- 自反性 ✓：(1,1),(2,2),(3,3) 都在 R 中
- 反对称性 ✓：需检查 (i,j)∈R 且 i≠j 时 (j,i)∉R
  - (1,2)∈R → (2,1)∉R ✓
  - (2,3)∈R → (3,2)∉R ✓
- 传递性 ✗：(1,2)∈R 且 (2,3)∈R → 需 (1,3)∈R，但 (1,3)∉R

故 R **不是**偏序关系。缺少 (1,3) 使其不满足传递性。✓

**例题5**：A={1,2,3,4,5,6} 上的模 3 同余关系 R：aRb ⇔ a≡b (mod 3)。求等价类。

**解**：
等价类按除以 3 的余数划分：
- [1] = {1, 4}（余 1）
- [2] = {2, 5}（余 2）
- [3] = {3, 6}（余 0）

商集 A/R = {{1,4}, {2,5}, {3,6}}，构成 A 的一个划分。✓

**例题6**：设 R₁={(a,b) | a>b}，R₂={(a,b) | a≤b} 是整数集上的关系。问 R₁ 和 R₂ 有什么性质？

**解**：
**R₁ (大于关系)**：
- 自反性：✗，a>a 永远不成立 → 反自反性 ✓
- 对称性：✗，a>b 不意味着 b>a → 反对称性 ✓（a>b 且 b>a 不可能）
- 传递性：✓，a>b 且 b>c ⇒ a>c

**R₂ (小于等于关系)**：
- 自反性：✓，a≤a 恒成立
- 对称性：✗，a≤b 不意味着 b≤a
- 反对称性：✓，a≤b 且 b≤a ⇒ a=b
- 传递性：✓，a≤b 且 b≤c ⇒ a≤c

R₂ 是**偏序关系**（且是**全序关系**）。✓

**例题7**：写出关系矩阵并判断传递性。A={a,b,c}，R={(a,a),(a,b),(b,c)}

**解**：
关系矩阵 M（行列顺序 a,b,c）：
```
     a  b  c
  a [1  1  0]
  b [0  0  1]
  c [0  0  0]
```

**传递性检查**：M² 表示长度为 2 的通路。
```
M² = M × M（布尔乘法）=
     a  b  c
  a [1  1  1]   ← (a,c)的出现因为 a→b→c
  b [0  0  0]
  c [0  0  0]
```

传递闭包的矩阵：若 M² 中某个位置为 1 而 M 中对应的位置为 0，则关系不传递。
M² 中 (a,c)=1 但 M 中 (a,c)=0 ⇒ **不满足传递性**。✓

## 5. 函数

### 5.1 基本概念

函数是特殊的二元关系：对每个 x∈A，**恰有一个** y∈B 使得 f(x)=y。

**函数的类型**：

| 类型 | 定义 | 条件 |
|------|------|------|
| 单射 (injective) | 不同元素映射到不同值 | f(x₁)=f(x₂) ⇒ x₁=x₂ |
| 满射 (surjective) | 陪域中每个元素都有原像 | ∀y∈B, ∃x∈A: f(x)=y |
| 双射 (bijective) | 既单射又满射 | 一一对应 |

**复合函数**：`(g∘f)(x) = g(f(x))`

**逆函数**：f 是双射时存在 f⁻¹，满足 f⁻¹∘f = Iₐ

---

### 解题示例

**例题1**：判断 f: R→R, f(x)=x² 是否是单射？满射？

**解**：
**单射检验**：f(1)=1, f(−1)=1，1≠−1 但 f(1)=f(−1)，不是单射 ✗

**满射检验**：对 y=−1，找不到 x∈R 使 x²=−1，不是满射 ✗

f(x)=x² 既不是单射也不是满射。✓

---

**例题2**：设 f(x)=2x+1, g(x)=x²，求 f∘g 和 g∘f

**解**：
- `(f∘g)(x) = f(g(x)) = f(x²) = 2x² + 1`
- `(g∘f)(x) = g(f(x)) = g(2x+1) = (2x+1)² = 4x² + 4x + 1`

注意：f∘g ≠ g∘f，复合运算**不满足交换律**。✓

---

**例题3**：f: R→R, f(x)=3x−2，判断是否双射，若是我求出 f⁻¹

**解**：
- **单射**：f(x₁)=f(x₂) ⇒ 3x₁−2=3x₂−2 ⇒ x₁=x₂ ✓
- **满射**：对任意 y∈R，取 x=(y+2)/3，有 f(x)=y ✓
- 故 f 是双射 ✓

求逆函数：
`y = 3x − 2` ⇒ `x = (y+2)/3` ⇒ `f⁻¹(x) = (x+2)/3` ✓

---

**例题4**：判断 f: Z→Z, f(x)=x+1 是否双射，求 f⁻¹ 和 f∘f

**解**：
- **单射**：f(x₁)=f(x₂) ⇒ x₁+1=x₂+1 ⇒ x₁=x₂ ✓
- **满射**：∀y∈Z，取 x=y−1，f(x)=y ✓
- 双射 ✓
- f⁻¹(x) = x−1 ✓
- f∘f(x) = f(x+1) = (x+1)+1 = x+2 ✓

---

**例题5**：设 f: {1,2,3,4} → {a,b,c} 定义为 f(1)=a, f(2)=b, f(3)=a, f(4)=b。问 f 是单射？满射？

**解**：
- |定义域|=4, |陪域|=3，由鸽巢原理不可能单射 ✗
- 检查满射：陪域中有 c 没有原像 → 不是满射 ✗
- 既不是单射也不是满射 ✓

---

**例题6**：证明：若 f 和 g 都是单射，则 f∘g 也是单射

**证**：
假设 (f∘g)(x₁) = (f∘g)(x₂)
即 f(g(x₁)) = f(g(x₂))
因为 f 是单射，所以 g(x₁) = g(x₂)
又因为 g 是单射，所以 x₁ = x₂

故 f∘g 是单射 ✓

---

**例题7**：f: R→R, f(x)=eˣ。分别判断限制陪域为 R 和 (0,+∞) 时的性质。

**解**：
- **f: R→R**：不是满射（无负数输出），是单射（严格单调增）
- **f: R→(0,+∞)**：是满射（∀y>0, x=ln y），是单射 → **是双射**，f⁻¹(x)=ln x ✓

## 6. 图论

### 6.1 基本概念

- **图 G=(V,E)**：V 是顶点集，E 是边集
- **无向图**：边是无序对 {u,v}
- **有向图**：边是有序对 (u,v)
- **度 d(v)**：与顶点 v 关联的边数（对有向图分入度 d⁻ 和出度 d⁺）

**握手定理**：`∑ d(v) = 2|E|`（度之和 = 边数的两倍）
- 推论：奇度顶点个数必为偶数

**特殊图**：
- 完全图 Kₙ：n 个顶点，每对不同顶点都有一条边（共 n(n−1)/2 条边）
- 二部图：顶点可分成两个集合，所有边连接不同集合中的顶点

### 6.2 通路与连通性

- **通路**：顶点-边交替序列 v₀e₁v₁e₂...eₖvₖ
- **简单通路**：边不重复
- **初级通路**：顶点不重复

**连通图**：任意两个顶点间存在通路

### 6.3 欧拉图与哈密顿图

| 类型 | 定义 | 充要条件 |
|------|------|---------|
| 欧拉通路 | 经过每条边恰好一次的通路 | 恰有0个或2个奇度顶点 |
| 欧拉回路 | 经过每条边恰好一次且回到起点 | 所有顶点度数为偶数 |
| 哈密顿通路 | 经过每个顶点恰好一次的通路 | 无充要条件(Ore定理等为充分条件) |

---

### 解题示例

**例题1**：设图 G 有 10 条边，4 个 3 度顶点，其余顶点度数不超过 2。问 G 至少有多少个顶点？

**解**：
设顶点数为 n。

由握手定理：`∑d(v) = 2×10 = 20`

已知 4 个 3 度顶点贡献 4×3 = 12，剩余度数 = 20−12 = 8。

其余 n−4 个顶点每个度数 ≤2，要使总度数达 8：
- 若每个都是 2，需要 `8/2 = 4` 个顶点
- 故最少总顶点数 = `4 + 4 = 8` ✓

---

**例题2**：判断下图的度数序列 `(1,1,2,3,4,5)` 是否能对应简单图

**解**：
用**Havel-Hakimi**算法：
1. 从大到小排序：`(5,4,3,2,1,1)`
2. 去掉首项 5，后面 5 项各减 1：`(3,2,1,0,0)`
3. 排序：`(3,2,1,0,0)`
4. 去掉首项 3，后面 3 项各减 1：`(1,0,−1,0)` ← 出现负数

**不可能**是简单图的度数序列。✓

---

**例题3**：K₅（5个顶点的完全图）是否有欧拉回路？

**解**：
K₅ 每个顶点的度数 = 4（偶数）
据欧拉回路充要条件，所有顶点的度数为偶数 ⇒ **有欧拉回路** ✓

> 例如回路：v₁→v₂→v₃→v₄→v₅→v₁→v₃→v₅→v₂→v₄→v₁

---

**例题4**：设无向图 G 有 7 个顶点，每个顶点的度至少为 3。证明 G 是连通图（假设无孤立点和多重边）。

**解**：
反证法：假设 G 不连通，则存在两个不连通的连通分支。

设较小的分支有 k 个顶点（1≤k≤3，因为总数 7，如果 k≥4 则另一个分支≤3，对称）。则该分支每个顶点度 ≤ k−1（因与其他分支无连接）。

若 k=1：该顶点度 = 0 < 3，与"每个顶点度≥3"矛盾 ✗
若 k=2：每个顶点度 ≤ 1 < 3，矛盾 ✗
若 k=3：每个顶点度 ≤ 2 < 3，矛盾 ✗

故 G 必连通。✓

---

**例题5**：设 G 是 6 个顶点的简单图，证明 G 和其补图中至少有一个是连通图。

**解**：
反证法：假设 G 和其补图都不连通。

取 G 中两个不连通分支中的顶点 u, v（无边的顶点对）。则在补图中 (u,v) 必为边。

类似推理会导致矛盾。更简洁的证明：
任取 G 中的一个顶点 v。若在 G 中与 v 不连通的其他顶点组成集合 S，则在补图中 v 与 S 中所有顶点相邻，故补图中 v 可到达 S 中所有顶点。

考虑 v 在 G 中所在的连通分支 V₁，以及 v 不连通的部分 V₂。在补图中：v 连接到 V₂ 的所有顶点，且 V₁ 中的顶点如果互相不连通则通过 v 中转。

可以严格证明补图必连通。（此例为经典结论，完整证明略。）✓

**例题6**：判断下面这个图是否为二部图：
```
边集：v₁−v₂, v₂−v₃, v₃−v₄, v₄−v₁, v₁−v₃
（5 条边围成的四边形 v₁−v₂−v₃−v₄−v₁ 加对角线 v₁−v₃）
```

**解**：
检查是否有奇圈：
- 圈 v₁→v₂→v₃→v₁ 长度为 3（奇圈！）

含有奇圈（三角形），故**不是二部图**。✓

> 定理：一个图是二部图 ⇔ 无奇圈。

**例题7**：7 个顶点的完全图 K₇ 有多少条边？是否存在欧拉回路？

**解**：
边数 = C(7,2) = 21 条 ✓

每个顶点的度 = 6（偶数），根据欧拉回路充要条件，所有顶点度数为偶数 ⇒ **存在欧拉回路**。✓

**例题8**：设图 G 有 9 个顶点，每个顶点的度数为 5 或 6。若度数为 5 的顶点有 4 个，求度数为 6 的顶点有多少个？G 有多少条边？

## 7. 树

### 7.1 基本概念

**树**：连通无回路的无向图。

**树的等价定义**（以下任一条都等价于 T 是树）：
1. T 连通且无回路
2. T 连通且 m = n−1（m边, n顶点）
3. T 无回路且 m = n−1
4. T 中任意两顶点间恰有一条初级通路

**生成树**：连通图的生成子图，是一棵树且包含所有顶点

**最小生成树 (MST)**：加权连通图中权值之和最小的生成树

### 7.2 二叉树

| 遍历方式 | 顺序 |
|---------|------|
| 先序遍历 | 根 → 左子树 → 右子树 |
| 中序遍历 | 左子树 → 根 → 右子树 |
| 后序遍历 | 左子树 → 右子树 → 根 |

---

### 解题示例

**例题1**：用 Kruskal 算法求下图的最小生成树。

```
边集（按权值升序）：
(1,6)=1, (2,3)=2, (4,5)=2, (1,2)=3, (3,4)=3,
(2,5)=4, (5,6)=5, (1,3)=6, (3,5)=7
```

| 步骤 | 选边 | 权值 | 是否成回路 |
|------|------|------|----------|
| 1 | (1,6) | 1 | 否 |
| 2 | (2,3) | 2 | 否 |
| 3 | (4,5) | 2 | 否 |
| 4 | (1,2) | 3 | 否 |
| 5 | (3,4) | 3 | 否 |
| — | (2,5) | 4 | 是（2-3-4-5 已连通）|
| — | (5,6) | 5 | 是（1-6和1-2-3-4-5已连通）|

已选 5 条边，n=6，满足 n−1=5，算法结束。

```

    6---1---2---3---4---5
```

**例题3**：证明一棵有 n 个顶点的树至少有 2 片叶子（n≥2）。

## 8. 组合数学

### 8.1 基本计数原理

**加法原理**：做一件事有 n 类方法，第 i 类有 mᵢ 种方法，总方法数 = m₁+m₂+...+mₙ

**乘法原理**：做一件事分 n 步，第 i 步有 mᵢ 种方法，总方法数 = m₁×m₂×...×mₙ

### 8.2 排列与组合

| 类型 | 公式 | 含义 |
|------|------|------|
| 排列 P(n,r) | `n!/(n−r)!` | n个不同元素取r个**有序**排列 |
| 组合 C(n,r) | `n!/[r!(n−r)!]` | n个不同元素取r个**无序**选择 |

常用恒等式：
- C(n,r) = C(n, n−r)（对称性）
- C(n,r) = C(n−1,r) + C(n−1,r−1)（Pascal恒等式）

### 8.3 鸽巢原理

**简单形式**：n+1 只鸽子放入 n 个鸽巢，至少有一个鸽巢中有 ≥2 只鸽子。

**广义形式**：q₁+q₂+...+qₙ−n+1 个物体放入 n 个盒子，则存在某个 i，第 i 个盒子至少有 qᵢ 个物体。

### 8.4 二项式定理

`(a+b)ⁿ = C(n,0)aⁿ + C(n,1)aⁿ⁻¹b + ... + C(n,k)aⁿ⁻ᵏbᵏ + ... + C(n,n)bⁿ`

---

### 解题示例

**例题1**：用数字 0,1,2,3,4,5 可以组成多少个没有重复数字的 4 位数？

**解**：
千位不能是 0，有 5 种选择（1~5中任选）
剩余 5 个数字中选 3 个排列到百十个位：P(5,3) = 5×4×3 = 60

总数 = `5 × 60 = 300` 个 ✓

---

**例题2**：从 10 人中选 5 人参加比赛，其中甲、乙两人至少选一人，有多少种选法？

**解**：
**方法一（补集法）**：
- 总选法：C(10,5) = 252
- 甲乙都不选的选法：C(8,5) = 56
- 所求 = `252 − 56 = 196` ✓

**方法二（分类法）**：
- 选甲不选乙：需从其余8人中选4人 = C(8,4) = 70
- 选乙不选甲：同上 = 70
- 甲乙都选：需从其余8人中选3人 = C(8,3) = 56
- 所求 = `70 + 70 + 56 = 196` ✓

---

**例题3**：证明任意 5 个整数中，必存在 3 个数之和能被 3 整除。

**解**（鸽巢原理）：
任何整数 mod 3 的余数为 0, 1, 2。

5 个数对 3 的余数情况：
- 若有 3 个数余数**相同**（同余类），这 3 个数之和 mod 3 = 3r ≡ 0 ✓
- 否则，每个余数类最多 2 个数。5 个数分入 3 类，由鸽巢原理，必有某类至少 ⌈5/3⌉=2 个数，且不可能有 3 个同余类
  - 此时必存在 3 个数余数分别为 0, 1, 2（因为 5 个分布在 3 类中，若缺少某类，其余两类总共最多 4 个但有一类最多 2 个，矛盾）
  - 这三个数之和 mod 3 = 0+1+2 = 3 ≡ 0 ✓

故无论哪种情况，必存在 3 个数之和被 3 整除。✓

---

**例题4**：将 10 个相同的球放入 4 个不同的盒子，允许空盒，有多少种放法？

**解**：
隔板法：将 10 个球用 3 个隔板分成 4 组。

允许空盒 → 等价于求 `x₁+x₂+x₃+x₄=10` 的非负整数解个数。

公式：C(n+k−1, k−1) = C(10+4−1, 4−1) = C(13, 3) = 286 ✓

---

**例题5**：展开 (2x−3y)⁵。

**解**：
用二项式定理：
```
(2x−3y)⁵ = C(5,0)(2x)⁵(−3y)⁰ + C(5,1)(2x)⁴(−3y)¹ + C(5,2)(2x)³(−3y)²
           + C(5,3)(2x)²(−3y)³ + C(5,4)(2x)¹(−3y)⁴ + C(5,5)(2x)⁰(−3y)⁵

= 1·32x⁵ + 5·16x⁴·(−3y) + 10·8x³·9y² + 10·4x²·(−27y³) + 5·2x·81y⁴ + 1·243y⁵
         （注意 (−3y) 的幂次符号交替）

= 32x⁵ − 240x⁴y + 720x³y² − 1080x²y³ + 810xy⁴ − 243y⁵
```
✓

---

**例题6**：求 (1+x)²⁰ 的展开式中 x¹⁰ 的系数。

**解**：
x¹⁰ 项 = C(20,10) × 1²⁰⁻¹⁰ × x¹⁰

系数 = C(20,10) = 184756 ✓

---

**例题7**：7 个人排成一排照相，其中 3 个女生必须相邻。有多少种排法？

**解**：
将 3 个女生视为一个整体"大元素"，问题化为 5 个元素（4 男 + 1"大元素"）的全排列：5! = 120

大元素内部 3 个女生也需排列：3! = 6

总排列数 = 5! × 3! = 120 × 6 = 720 ✓

---

**例题8**：从 5 个数字 {1,2,3,4,5} 中可重复地取 3 个，组成一个 3-组合（顺序无关），有多少种取法？

**解**：
可重复组合数公式：H(n,r) = C(n+r−1, r) = C(5+3−1, 3) = C(7, 3) = 35 ✓

也可理解为：x₁+x₂+x₃+x₄+x₅=3 的非负整数解个数（隔板法），结果一致。✓

---

**例题9**：证明 C(n,0)+C(n,1)+...+C(n,n) = 2ⁿ

**证**：
在二项式定理 `(1+x)ⁿ = ∑C(n,k)xᵏ` 中取 x=1：
(1+1)ⁿ = ∑C(n,k)×1ᵏ = ∑C(n,k)
即 2ⁿ = C(n,0)+C(n,1)+...+C(n,n) ✓

## 9. 代数系统

### 9.1 基本概念

**代数系统**：集合 S 及定义在 S 上的若干运算构成的系统。

**二元运算 *** 的性质：

| 性质 | 定义 |
|------|------|
| 封闭性 | ∀a,b∈S, a*b∈S |
| 结合律 | (a*b)*c = a*(b*c) |
| 交换律 | a*b = b*a |
| 单位元 e | a*e = e*a = a |
| 逆元 a⁻¹ | a*a⁻¹ = a⁻¹*a = e |
| 分配律 | a*(b∘c) = (a*b)∘(a*c) |

### 9.2 群、环、域

**半群**：非空集合 + 满足结合律的二元运算

**群 (Group)**：半群 + 存在单位元 + 每个元素有逆元

**Abel 群（交换群）**：群 + 运算满足交换律

**环 (Ring)**：集合 S 上有两个二元运算 +（加法）和 ×（乘法）：
- (S,+) 是 Abel 群
- (S,×) 是半群
- 乘法对加法满足分配律

**域 (Field)**：环 + (S−{0}, ×) 也是 Abel 群（即非零元都有乘法逆元）

---

### 解题示例

**例题1**：判断 (Z, +) 是否是群，若是，是否交换？

**解**：
- **封闭性** ✓：整数相加仍为整数
- **结合律** ✓：(a+b)+c = a+(b+c)
- **单位元**：0，因为 a+0=0+a=a ✓
- **逆元**：a 的逆元是 −a，因为 a+(−a)=0 ✓
- **交换律**：a+b = b+a ✓

故 (Z, +) 是群，且是 **Abel 群**。✓

---

**例题2**：集合 G={1, −1, i, −i} 关于复数乘法是否构成群？

**解**：
验证封闭性：
```
1×1=1  1×(−1)=−1  1×i=i  1×(−i)=−i
(−1)×(−1)=1  (−1)×i=−i  (−1)×(−i)=i
i×i=−1  i×(−i)=1
(−i)×(−i)=−1
```
运算封闭 ✓

- 结合律 ✓（复数乘法满足结合律）
- 单位元 1 ✓
- 逆元：1⁻¹=1, (−1)⁻¹=−1, i⁻¹=−i, (−i)⁻¹=i ✓
- 交换律 ✓（复数乘法可交换）

故 G 是 **Abel 群**。这也是 4 阶循环群。✓

---

**例题3**：设 G={2ⁿ | n∈Z}（即所有 2 的整数次幂），运算为普通乘法。证明 G 是群。

**解**：
- 封闭性：2ᵐ × 2ⁿ = 2ᵐ⁺ⁿ ∈ G ✓
- 结合律：乘法满足结合律 ✓
- 单位元：2⁰ = 1 ∈ G ✓
- 逆元：(2ⁿ)⁻¹ = 2⁻ⁿ ∈ G ✓
- 交换律：2ᵐ2ⁿ = 2ⁿ2ᵐ ✓

G 是 Abel 群。且 G 与 (Z,+) 同构（映射 φ(2ⁿ)=n）。✓

---

**例题4**：设 H 是群 G 的子集，H={e}（仅含单位元）。证明 H 是 G 的子群。

**证**：
子群判定条件（只需检查封闭性和逆元存在）：
- 封闭：e·e = e ∈ H ✓
- 逆元：e⁻¹ = e ∈ H ✓
故 H={e} 是 G 的子群（称为平凡子群）。✓

---

**例题5**：判断 (Z, +, ×) 是否是环？是否是域？

**解**：
**(Z, +, ×) 是否是环？**
- (Z,+) 是 Abel 群 ✓（前面已证）
- (Z,×) 是半群（封闭且满足结合律）✓
- 乘法对加法满足分配律 ✓

故 (Z, +, ×) 是**环**。✓

**(Z, +, ×) 是否是域？**
- 域要求 (Z−{0}, ×) 也是 Abel 群
- 检查：2 的乘法逆元是 1/2 ∉ Z

故 (Z, +, ×) 不是域。✓

---

**例题6**：设群 G={e, a, b, c}，运算表如下，判断是否是群，单位元是谁。

```
* | e  a  b  c
--+-----------
e | e  a  b  c
a | a  e  c  b
b | b  c  e  a
c | c  b  a  e
```

**解**：
- 封闭性：查表可见，任意两元素运算结果仍在 G 中 ✓
- 结合律：需逐个验证，因篇幅省略（可通过对称性推断）
- 单位元：看 e 行/列，e*x = x*e = x → e 是单位元 ✓
- 逆元：每个元素都有逆元
  - e⁻¹=e（e*e=e）
  - a⁻¹=a（a*a=e）
  - b⁻¹=b（b*b=e）
  - c⁻¹=c（c*c=e）
  每个元素都是自己的逆元 ✓

G 是群。这是 **Klein 四元群**。✓

---

**例题7**：证明在有限半群中，必存在幂等元（即 ∃a, a²=a）。

**证**：
设 G 是有限半群，|G|=n。任取 x∈G，考虑序列 x, x², x³, ..., xⁿ⁺¹。

共 n+1 个元素，但 G 只有 n 个元素。由鸽巢原理，存在 i<j 使得 xⁱ = xʲ。

令 k = j−i > 0，则 xⁱ = xⁱ⁺ᵏ = xⁱ·(xᵏ)。

反复用此等式，可得 xⁱ = xⁱ⁺ᵐᵏ 对任意 m≥0 成立。

取 m 足够大使 mk ≥ i，令 a = xᵐᵏ，则 a² = x²ᵐᵏ。由 xⁱ = xⁱ⁺ᵐᵏ 可得 x²ᵐᵏ = xᵐᵏ，即 a² = a。

故存在幂等元 a = xᵐᵏ。✓

---

## 10. 递推关系

### 10.1 基本概念

**递推关系**：用序列前面项表示后面项的等式。

**常系数线性齐次递推关系**：形式为
`aₙ = c₁aₙ₋₁ + c₂aₙ₋₂ + ... + cₖaₙ₋ₖ`

**特征方程**：`rᵏ − c₁rᵏ⁻¹ − c₂rᵏ⁻² − ... − cₖ = 0`

### 10.2 通解形式

以二阶 `aₙ = c₁aₙ₋₁ + c₂aₙ₋₂` 为例，特征方程 `r² − c₁r − c₂ = 0`：

| 根的情况 | 通解形式 |
|---------|---------|
| 两不等实根 r₁,r₂ | aₙ = αr₁ⁿ + βr₂ⁿ |
| 重根 r（二重） | aₙ = (α + βn)rⁿ |
| 共轭复根 | aₙ = ρⁿ(α cos nθ + β sin nθ) |

其中 α, β 由初始条件确定。

---

### 解题示例

**例题1**：求解递推关系 `aₙ = 3aₙ₋₁ − 2aₙ₋₂`，a₀=2, a₁=3

**解**：
特征方程：`r² − 3r + 2 = 0`

`(r−1)(r−2) = 0`，根为 r₁=1, r₂=2（两不等实根）

通解：`aₙ = α·1ⁿ + β·2ⁿ = α + β·2ⁿ`

代入初始条件：
```
n=0: a₀ = α + β = 2
n=1: a₁ = α + 2β = 3
```
解得：β=1, α=1

∴ `aₙ = 1 + 2ⁿ` ✓

验证：a₂ = 1+4=5，由递推 a₂=3a₁−2a₀=9−4=5 ✓

---

**例题2**：Fibonacci 数列 `Fₙ = Fₙ₋₁ + Fₙ₋₂`，F₀=0, F₁=1，求通项公式。

**解**：
特征方程：`r² − r − 1 = 0`

根：`r₁ = (1+√5)/2`（黄金比例 φ），`r₂ = (1−√5)/2`

通解：`Fₙ = α·φⁿ + β·((1−√5)/2)ⁿ`

代入初始条件：
```
n=0: α + β = 0  → β = −α
n=1: αφ + β((1−√5)/2) = 1
```
代入 β=−α：
`αφ − α((1−√5)/2) = α[φ − (1−√5)/2] = α√5 = 1`

α = 1/√5，β = −1/√5

∴ `Fₙ = (φⁿ − ((1−√5)/2)ⁿ) / √5` ✓

---

**例题3**：求解 `aₙ = 4aₙ₋₁ − 4aₙ₋₂`，a₀=1, a₁=3

**解**：
特征方程：`r² − 4r + 4 = 0`

`(r−2)² = 0`，重根 r=2

通解：`aₙ = (α + βn)·2ⁿ`

代入初始条件：
```
n=0: (α + 0)·1 = 1  → α = 1
n=1: (1 + β)·2 = 3  → 1+β = 3/2  → β = 1/2
```

∴ `aₙ = (1 + n/2)·2ⁿ = 2ⁿ + n·2ⁿ⁻¹` ✓

验证：a₂ = 4+4=8，由递推 a₂=4a₁−4a₀=12−4=8 ✓

---

**例题4**：求解 `aₙ + aₙ₋₁ − 6aₙ₋₂ = 0`，a₀=1, a₁=0

**解**：
特征方程：`r² + r − 6 = 0`

`(r+3)(r−2) = 0`，根为 r₁=−3, r₂=2

通解：`aₙ = α·(−3)ⁿ + β·2ⁿ`

代入初始条件：
```
n=0: α + β = 1
n=1: −3α + 2β = 0  →  β = 3α/2
```
代入第一式：α + 3α/2 = 1 → 5α/2 = 1 → α = 2/5, β = 3/5

∴ `aₙ = (2/5)·(−3)ⁿ + (3/5)·2ⁿ` ✓

---

**例题5**：求解递推关系 `aₙ = 2aₙ₋₁ − aₙ₋₂`，a₀=1, a₁=2

**解**：
特征方程：`r² − 2r + 1 = 0`

`(r−1)² = 0`，重根 r=1

通解：`aₙ = (α + βn)·1ⁿ = α + βn`

代入初始条件：
```
n=0: α = 1
n=1: α + β = 2  → β = 1
```

∴ `aₙ = 1 + n` ✓（这就是等差数列！）

---

**例题6**：求解非齐次递推关系 `aₙ = 2aₙ₋₁ + 1`，a₀=0

**解**：
这是**非齐次**一阶递推关系。

**方法一（迭代法）**：
a₁ = 2·0 + 1 = 1
a₂ = 2·1 + 1 = 3
a₃ = 2·3 + 1 = 7
a₄ = 2·7 + 1 = 15

猜测：aₙ = 2ⁿ − 1

**验证**：aₙ₋₁ = 2ⁿ⁻¹−1，则 2aₙ₋₁+1 = 2(2ⁿ⁻¹−1)+1 = 2ⁿ−2+1 = 2ⁿ−1 = aₙ ✓

**方法二（齐次+特解）**：
齐次通解：aₙ⁽ʰ⁾ = α·2ⁿ
特解（常数）aₙ⁽ᵖ⁾ = C，代入：C = 2C+1 → C = −1
通解 = α·2ⁿ − 1

代入 a₀=0：α − 1 = 0 → α = 1
∴ aₙ = 2ⁿ − 1 ✓

---

**例题7**：求带有复根的递推关系 `aₙ = 2aₙ₋₁ − 2aₙ₋₂`，a₀=0, a₁=1 的通解

**解**：
特征方程：`r² − 2r + 2 = 0`

根：r = (2 ± √(4−8))/2 = (2 ± 2i)/2 = 1 ± i

写为极式：`1+i = √2(cos(π/4) + i sin(π/4))`，ρ=√2, θ=π/4

通解：`aₙ = (√2)ⁿ (α cos(nπ/4) + β sin(nπ/4))`

代入初始条件：
```
n=0: α·cos 0 + β·sin 0 = α = 0  → α = 0
n=1: √2(α cos(π/4) + β sin(π/4)) = √2(0 + β/√2) = β = 1  → β = 1
```

∴ `aₙ = (√2)ⁿ sin(nπ/4)` ✓

---

**例题8**：用生成函数法求 `aₙ = 3aₙ₋₁ − 2aₙ₋₂`，a₀=2, a₁=3 的通解

**解**：
设 G(x) = ∑ₙ₌₀∞ aₙxⁿ

递推式两边乘 xⁿ 并求和（n≥2）：
```
∑ₙ₌₂∞ aₙxⁿ = 3∑ₙ₌₂∞ aₙ₋₁xⁿ − 2∑ₙ₌₂∞ aₙ₋₂xⁿ
```
`G(x) − a₀ − a₁x = 3x(G(x) − a₀) − 2x²G(x)`

代入 a₀=2, a₁=3：
```
G(x) − 2 − 3x = 3xG(x) − 6x − 2x²G(x)
G(x)(1 − 3x + 2x²) = 2 − 3x
```
```
G(x) = (2−3x) / (1−3x+2x²) = (2−3x) / [(1−x)(1−2x)]
```
部分分式展开：G(x) = A/(1−x) + B/(1−2x)

A(1−2x) + B(1−x) = 2−3x
x=1: A(−1) = −1 → A = 1
x=1/2: B(1/2) = 2−3/2 = 1/2 → B = 1

G(x) = 1/(1−x) + 1/(1−2x) = ∑xⁿ + ∑(2x)ⁿ = ∑(1+2ⁿ)xⁿ

∴ aₙ = 1 + 2ⁿ ✓（与特征方程法一致）
