## 使用说明

每谈一项，确认以下三个问题都有了明确答案，才算谈完：

- **包含什么？**（写进报价单的每一项）
- **不包含什么？**（最容易漏掉的增项来源）
- **超出约定范围怎么算？**（一口价还是按实结算？单价多少？）

以下每一节都是你在签约前必须和设计师/销售逐条确认的内容。对方答不上来或者含糊其辞=换公司。

## 一、总体报价方式

### 1.1 报价模式确认

<div style="overflow-x:auto;">

| 问题 | 你要得到的回答 | 红线 |
|------|------|------|
| 是套餐报价还是清单报价？ | 清单报价（每一项单独列出） | 套餐报价（699/899/999/㎡）— 不签 |
| 最终结算价能否写入合同？ | "一口价，超出5%由我方承担" | "以实际用量为准" — 不签 |
| 这个价格是含税价格吗？ | 含税，给你开票 | 不含税，后期自己补税点 |


</div>
### 1.2 必须要求的分项报价

让对方出具一份**按房间×按项目**拆分的报价表：

<div style="overflow-x:auto;">

| 空间 | 拆改 | 水电 | 防水 | 贴砖 | 吊顶 | 墙面 | 其他 |
|------|------|------|------|------|------|------|------|
| 客厅+餐厅 | | | | | | | |
| 厨房 | | | | | | | |
| 主卧 | | | | | | | |
| 次卧 | | | | | | | |
| … | | | | | | | |


</div>
每格写**材料费+人工费+管理费**分别多少。

**没有这样一份报表 = 不签。**

## 二、拆改（谈判5项）

### 2.1 拆墙

<div style="overflow-x:auto;">

| 谈什么 | 要谈到什么程度 |
|------|------|
| 包含哪些墙？ | "拆墙明细表"——每一面标注位置、面积、厚度（12墙/24墙）、单价 |
| 不包含的墙 | 如果你们自己额外想拆的怎么算——单价___元/㎡ |
| 承重墙检测 | 由乙方提供原始结构图明确标注承重墙位置，拆错责任全部乙方承担，含恢复和罚款 |
| 铲墙皮 | 是否包含？铲到什么程度？写死"铲至水泥砂浆层，不留白灰" |
| 垃圾清运 | 包含在总价内？运到哪里？运多少次？ |


</div>
### 2.2 砌墙

<div style="overflow-x:auto;">

| 谈什么 | 要谈到什么程度 |
|------|------|
| 包含哪些新砌墙？ | "砌墙明细表"——每面墙位置、长×高、厚度、单价 |
| 材料用什么？ | 砖(加气块/红砖)、水泥(品牌/标号)、砂浆比例 |
| 植筋包含吗？ | "每50cm植一根钢筋"，含在报价内（如果谈成另加钱=坑） |
| 钢丝网包含吗？ | "新旧墙交接处挂钢丝网宽度≥20cm"，含在报价内 |
| 地梁包含吗？ | "卫生间新砌墙倒地梁高≥20cm"，含在报价内 |
| 门洞过梁包含吗？ | 含在报价内，材质（钢筋混凝土/槽钢） |


</div>
### 2.3 拆改最容易出现的增项

下面这些项目是装修公司最喜欢在拆改阶段加钱的：

<div style="overflow-x:auto;">

| 增项 | 预防 |
|------|------|
| "这面墙比你想象的长，多拆了X平米" | 签合同前要求量出拆墙面积，定死总价 |
| "原来这里还有一个凸出物需要打掉" | 量房时设计师应全部记录，漏记=他全责 |
| "垃圾太多，要多拉一车" | 写清合同含垃圾清运，不限车数，一价全包 |
| "这里拆除后发现需要加固" | 这是乙方应该预判的，算他自己没量清楚 |
| 旧门窗拆除 | 明确：□含 □不含，不含的话多少一套 |


</div>
**谈完拆改，合同里写清：**
> 报价单内的拆改数量为 ___㎡拆墙、___㎡砌墙、___㎡铲墙皮。此部分一口价，最终结算不再补差。

## 三、水电改造（谈判最重要的一项，15项逐条过）

### 3.1 计价方式

<div style="overflow-x:auto;">

| 谈什么 | 要谈到什么程度 |
|------|------|
| 水电是一口价还是按实结算？ | **必须一口价**。不接受"按实结算"。量房时就应确定了全部点位，不应该有"不知道"。 |
| 一口价包含多少个点位？ | 合同里写：全屋___个插座 + ___个开关 + ___个冷水 + ___个热水 + ___个排水 + ___个网线 + ___个电视线 = 总计___点位 |
| 超出点位的单价 | 每增一个插座___元、开关___元、网线___元、水管___元。这些单价现在写在合同里 |
| 如果实际点位比报价少 | 按单价退还（这条写上，对方才不敢多估） |


</div>
### 3.2 电线材料

<div style="overflow-x:auto;">

| 谈什么 | 要谈到什么程度 |
|------|------|
| 电线品牌型号 | 具体到品牌、系列。如"德力西 BV 硬线"、"熊猫 BV"。不能只写"国标" |
| 各路线规格 | 照明___mm² / 普通插座___mm² / 厨卫___mm² / 空调___mm² / 入户___mm²。全部写死 |
| 强电箱回路数 | 全屋___路。回路清单：冰箱1路、空调各1路、厨卫各1路…（列在合同附件里） |
| 漏电保护器 | 每个回路用什么品牌的漏保，规格，现在定死 |
| 空开品牌和规格 | □施耐德<br>□西门子<br>□ABB<br>□正泰 + 具体型号 |


</div>
### 3.3 水管材料

<div style="overflow-x:auto;">

| 谈什么 | 要谈到什么程度 |
|------|------|
| 水管品牌 | □伟星<br>□日丰<br>□中财<br>□保利<br>□金牛，写具体 |
| 冷热水管 | 全部用热水管标准（壁厚≥3.4mm）。不能用冷水管标准 |
| 水管走顶还是走地？ | 明确写死。推荐"厨卫水管走顶，其余可走地" |
| 走顶的冷水管包保温棉 | □含 □不含。含=含在报价内 |
| 排水管用隔音棉包裹 | □含 □不含。主管必须包，报价含 |
| 前置过滤器 | □含 □不含。含的话品牌型号 |


</div>
### 3.4 具体内容

<div style="overflow-x:auto;">

| 谈什么 | 要谈到什么程度 |
|------|------|
| 强弱电箱是否更换？ | □更换 □不换。更换的话品牌/型号/回路数 |
| 强弱电交接处包锡纸 | 含在报价内 |
| 底盒品牌 | □伟星<br>□联塑<br>□公牛 |
| 网线 | □超五类<br>□六类<br>□屏蔽<br>□非屏蔽 |


</div>
### 3.5 水电最容易出现的增项

<div style="overflow-x:auto;">

| 增项 | 怎么预防 |
|------|------|
| "你家的线管比预计长" | 要求一口价，不接受按米结算 |
| "这个点位报价里没含" | 签合同前拿着点位清单一个一个确认 |
| "这个管子要加钱" | 包管、隔音棉这些明确是否含在报价里 |
| "配电箱太小装不下" | 设计师应该提前确定回路数量 → 确定箱体大小 |
| "你家墙上不能走管太长" | 承重墙开横槽≤30cm是规范，不是"加钱"的理由 |


</div>
**谈完水电，要求写入合同：**
> 水电改造为一口价承包。报价单所列点位已含全部水电内容（含强弱电箱、空开漏保、全部管线、包隔音棉、冷热水管全部热水管标准、强弱电交叉锡纸、底盒保护盖），最终结算不补差。

## 四、泥瓦（防水+贴砖+地面，谈判12项）

### 4.1 防水

<div style="overflow-x:auto;">

| 谈什么 | 要谈到什么程度 |
|------|------|
| 防水涂料品牌系列 | 品牌(□雨虹<br>□德高<br>□西卡) + 类型(□柔韧型/通用型)。卫生间用柔韧型 |
| 刷几遍？ | ≥2遍（纵横各一遍）。写进合同 |
| 哪些区域做防水？ | 卫地面+墙面、厨地面+墙面30cm、阳地面。写全 |
| 防水高度 | 淋浴≥1.8m / 洗手台≥1.2m / 其余≥30cm |
| 闭水试验多久？ | 48小时。含去楼下邻居家检查 |
| 管根/阴角附加层 | 含堵漏王+R角+附加层，报价含 |


</div>
### 4.2 瓷砖——主材部分

<div style="overflow-x:auto;">

| 谈什么 | 要谈到什么程度 |
|------|------|
| 每个空间的瓷砖 | 品牌(____)、系列(____)、型号(____)、规格(____×____mm)、色号(____#)。全部写进01合同附件 |
| 如果选不到满意的怎么办？ | 可以自己买吗？可以加钱升级吗？升级补差怎么算？ |
| 升级补差 | 比如800×800换成600×1200，每片补___元。现在就谈好公式 |
| 损耗率 | 5%。超出损耗的部分由乙方承担 |
| 补货 | 如果同一批次买不到怎么办？"乙方承诺同一空间使用同一批次瓷砖，如需补货由乙方负责调同批次色号" |
| 厨房墙面/地面/花砖/波导线/门槛石 | 地砖上墙是否需要额外费用？花砖一片多少钱？ |
| 阳角处理 | □海棠角 □阳角条，含在报价内 |


</div>
### 4.3 瓷砖——施工部分

<div style="overflow-x:auto;">

| 谈什么 | 要谈到什么程度 |
|------|------|
| 贴砖人工 | 不同规格瓷砖人工不同（300×600 vs 600×1200 vs 750×1500）。每种规格单价分别多少？ |
| 瓷砖胶/背胶 | 大砖上墙需要瓷砖胶+背胶。含在报价内吗？品牌型号？ |
| 美缝 | □含在报价内 □不含。含的话品牌色号。不含的话谁做、多少钱 |
| 墙地对缝 | □含 □不含。含=人工多加钱吗？ |


</div>
### 4.4 地面找平

<div style="overflow-x:auto;">

| 谈什么 | 要谈到什么程度 |
|------|------|
| 哪些区域做找平？ | 铺地板的区域必须找平（含在报价内） |
| 找平标准 | 2米靠尺≤3mm |


</div>
### 4.5 泥瓦最容易出现的增项

<div style="overflow-x:auto;">

| 增项 | 怎么预防 |
|------|------|
| "这个砖太大了，人工翻倍" | 签合同前确认全部瓷砖规格，按规格定人工费 |
| "你选的砖是特价款外的，要补差" | 选砖时当场确认哪些在套餐内哪些不在，写在纸上 |
| "背胶不含的，要另加" | 大砖上墙的瓷砖胶+背胶写进报价 |
| "你家的墙不平！要先找平再贴" | 量房时设计师应该看出来墙有多不平，现在报价就要含 |
| "烟道要包" | 含在报价内 |


</div>

<div style="overflow-x:auto;">

## 六、油漆（谈判10项）

<div style="overflow-x:auto;">

| 谈什么 | 要谈到什么程度 |
|------|------|
| 墙固/界面剂 | 品牌(____)，含在报价内 |
| 腻子品牌 | 品牌(____)，必须"耐水腻子"，不能"普通腻子" |
| 腻子遍数 | ≥3遍，含在报价内 |
| 乳胶漆品牌系列 | 品牌(____) + 系列(____)，不能只写"立邦" |
| 底漆 | □含 □不含。含的话品牌、遍数（1遍） |
| 面漆遍数 | 2遍，含在报价内 |
| 颜色 | 几面墙做颜色？几个颜色？包含吗？加一种颜色加___元 |
| 调色 | 机调还是手调？必须机调一次成型，不能分两次调 |
| 局部挂网 | 新旧墙交接+线槽处挂网，含在报价内 |
| 石膏板接缝 | 贴牛皮纸/网格带，含在报价内 |


</div>
**油漆最容易出现的增项：**
- "你要彩色漆！要加钱！" — 签合同前谈好哪几面墙是彩色的，是否含在报价内
- "全屋挂网吧，不然开裂" — 全屋挂网没必要，只局部挂。全屋挂加钱=忽悠
- "用进口漆吧，更好" — 签合同就定好漆，不临时升级
- "这个颜色刷不出来的，要换漆" — 签合同就定了颜色和漆的品牌

## 七、全屋定制（谈判最复杂，20项逐条过）

### 7.1 计价方式

<div style="overflow-x:auto;">

| 谈什么 | 要谈到什么程度 |
|------|------|
| 投影面积还是展开面积？ | □投影面积。建议投影面积。如对方用展开面积，不签 |
| 投影面积含什么？ | 必须明确：□柜门<br>□背板(___mm厚)<br>□层板<br>□挂衣杆<br>□收口条<br>□拉手。每项都要勾选 |
| 投影面积不含什么？ | 逐项确认：抽屉超送___个后加___元/个；玻璃门加___元/㎡；灯带加___元/m；反弹器加___元/个；见光面加___元/㎡；拉直器加___元/根… |
| 这些不含项的单价 | **必须现在就定下来**，签字后才定=想定多少定多少 |


</div>
### 7.2 板材

<div style="overflow-x:auto;">

| 谈什么 | 要谈到什么程度 |
|------|------|
| 柜体板品牌 | □万华禾香板<br>□兔宝宝<br>□莫干山<br>□千年舟<br>□爱格<br>□克诺斯邦<br>□露水河<br>□大亚 |
| 柜体板环保等级 | □ENF<br>□F☆☆☆☆<br>□E0。建议ENF。合同写死 |
| 柜体板厚度 | 18mm（非16mm）。合同写死 |
| 背板厚度 | 9mm（非5mm/3mm）。合同写死 |
| 背板颜色 | 与柜体同色。合同写死（防止用二手余料乱拼） |
| 柜门板材质 | 颗粒板/欧松板/密度板/PET/亚克力/烤漆 |
| 柜门板厚度 | 18mm/22mm/25mm。合同写死 |
| 门板表面处理 | 双饰面/PET/烤漆/亚克力。具体到系列和纹理 |
| 柜门颜色 | 合同附件写上色号。选色时拍照留档 |
| 封边工艺 | □PUR<br>□激光<br>□EVA。建议PUR以上。合同写死（PUR封边是胶水液态封，EVA是热熔胶封，PUR防潮好很多） |

</div>

### 7.4 各空间定制细节

## 十三、签合同前最后一步 —— 交叉验证

拿到对方最终的合同、报价、材料清单后，做这几件事：

1. **三份文件交叉比对** — 合同里提到的材料 → 附件清单里有没有？报价单里的项目 → 合同里有没有？三份文件互相印证，一处矛盾=不签

2. **对比至少3家** — 把A/B/C三家公司的报价表并列对比，看：
   - 哪家漏项了（报价表比别人少一页 = 漏项 = 后期增项）
   - 同一项目的材料品牌是否一致（品牌不一样=价差）

3. **打电话问一个"已完工半年以上"的业主** — 找公司要一个已完工半年的老客户联系方式，打电话问：
   - 竣工验收后有没有发现新问题？
   - 有问题了公司上门及时吗？
   - 入住半年了有什么后悔的？

4. **问清楚工地负责人是谁** — 签合同的设计师和实际施工的工长是不是同一个人？有没有工长的联系方式？工长同时管几个工地？

5. **再走一遍工地** — 签合同前再去一次他们正在施工的工地，看：
   - 现场是否整洁（干净=管理好）
   - 材料堆放是否规范（材料乱扔=不珍惜你家材料）
   - 问现场工人"老板结工资及时吗"（工人说老板好=公司靠谱，工人说拖欠=小心卷款跑路）
