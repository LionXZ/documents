## 第一阶段：选公司（不要只看样板间）

### 1.1 去哪里找装修公司

| 渠道 | 优缺点 | 建议 |
|------|------|------|
| 朋友/邻居/同事推荐 | 真实，但朋友家好不等于你家好 | 首选，但也要去过工艺 |
| 小区扫楼看工地 | 最真实，直接看同户型在做工地的工艺 | **强烈推荐**，下楼就能看 |
| 第三方平台（土巴兔/齐家网） | 名义上有保障，实则平台抽成导致增项 | 只当信息渠道，不全信 |
| 本地建材市场门口的小公司 | 价格低但没保障 | 不推荐新手 |
| 品牌连锁（业之峰/东易日盛/名雕等） | 品牌大但贵，且是加盟=换了个店名 | 恩施本地连锁不多，谨慎 |

### 1.2 初步筛选（加微信前就做）

1. **天眼查/企查查** — 查公司工商信息：
   - 注册时间（低于2年的慎重）
   - 有无官司纠纷、行政处罚
   - 法人是否频繁变更
   - 注册地址和实际办公地址是否一致
2. **裁判文书网** — 搜"公司名 + 装修合同"，查有没有打过官司
3. **抖音/小红书搜"公司名 + 投诉""公司名 + 踩坑"** — 本地圈子小，有人避雷就会发
4. **要求对方发正在做的工地地址** — 不能发的直接pass

### 1.3 进店考察（多注意细节）

- 进店不要留电话（留了就被反复骚扰）
- 不要被样板间迷惑，直接要去看在建工地
- 问他：**设计师会不会全程跟单？工长同时管几个工地？**
- 要求看已完工半年的案例照片（不要效果图，要实拍图），看有没有开裂/空鼓/门下沉
- 感受一下和设计师的沟通：他没听懂你说的话、一个劲推套餐 = 不靠谱

### 1.4 量房&出方案

- 让3-4家公司同时量房（同一天约到，让他们互相看见）
- 量房后要求出：
  - 平面布局图（2-3种方案供选择）
  - 拆改示意图
  - 初步报价单
- 如果某家公司量房后催你交定金，直接拒绝。"量房免费"是行业惯例。

## 第二阶段：谈价（分三轮打）

### 2.1 不谈套餐，要清单报价

**绝对不签任何"套餐"合同。**

- "699/㎡""899/㎡""999/㎡"套餐 = 基础款 + 所有好看的都加钱
- 要求出具**分项清单报价**：每项施工内容、材料品牌型号、单位、单价、数量
- 如果对方说"出不了这么细"——换一家，能出的公司多的是

### 2.2 第一轮：了解行情

拿到3-4份报价单后，不管价格高低，先做横向对比：

| 对比维度 | 怎么看 |
|------|------|
| 总价对比 | 去掉最高和最低，看中间两家 |
| 水电报价方式 | 是"一口价"还是"按实结算"？后者=坑 |
| 材料品牌 | A公司用伟星，B公司用杂牌？品牌差异=价差 |
| 项目完整性 | 有无漏项？（防水、垃圾清运、美缝、包管……） |
| 柜体报价方式 | 投影面积还是展开面积？含什么不含什么？|

### 2.3 第二轮：压价

核心话术——
- "B公司比你便宜6000块，也是用的伟星管+雨虹防水"
- "C公司报了XXX一平，换成你家这价我找你签"
- 不要只说"贵了"，要给具体差距数字

可以砍的幅度：恩施本地报价水分约**15%-25%**。

### 2.4 第三轮：不降价就升级

第二轮砍到底后，对方说"不能再降了"——那就升级材料：

话术——
- "总价我不降了，你帮我把瓷砖升级成东鹏/马可波罗"
- "你把柜体板换成万华禾香板 ENF 级"
- "你把铰链从东泰升级成百隆"
- "你再送我一个厨房台下盆 + 全屋美缝"

这是最后一轮。谈到这里如果对方拒绝一切升级——那就真到底了。

### 2.5 最佳签单时机

- **月底、季度末、年底**（12月）——销售冲业绩，更有弹性
- **3-4月、9-10月**——装修旺季前，公司愿意放价吸引订单
- **避开春节后1-2月**——工人都没回来，签了也是等
- 不要被"今天签才能享受这个价"吓住——好的价格下周也在，急着催你签的必有猫腻

## 第三阶段：签合同（最长的章节，每一条都关乎你的钱）

### 3.1 合同要包含的文件清单

签合同时必须有下列文件，缺一个都不签：

| 序号 | 文件 | 是否齐全 |
|------|------|------|
| 1 | 装修合同正本（建议用住建局标准合同模板） | □ |
| 2 | 材料清单附件（就是本文档"01"那一份） | □ |
| 3 | 全套施工图纸（不少于15张） | □ |
| 4 | 3D效果图（至少客餐厅+主卧） | □ |
| 5 | 报价清单（分项，含施工费+材料费明细） | □ |
| 6 | 工期进度表（甘特图） | □ |
| 7 | 付款节点表 | □ |
| 8 | 全屋定制产品清单（同上01文档第三部分） | □ |
| 9 | 质保承诺书（含质保时限和响应时间） | □ |
| 10 | 营业执照复印件（盖公章） | □ |

### 3.2 施工图纸最少包含

- 原始结构图
- 拆墙示意图
- 砌墙示意图
- 平面布置图
- 家具尺寸图
- 地面铺贴图（含铺贴方向+起铺点）
- 天花板吊顶图（含灯具定位）
- 开关布置图
- 插座布置图
- 水路走向图（冷热水分开标注）
- 立面索引图
- 各空间立面图（每个房间四面墙的立面）
- 节点大样图（特殊工艺的剖面详图）
- 全屋定制设计图（每个柜体的内部结构+外立面）

**图纸注明：** 经甲乙双方签字确认的图纸，任何一方不得擅自更改。如需变更，须办理书面变更手续。

### 3.3 工期条款

```
开工日期：20____年____月____日
竣工日期：20____年____月____日
总工期：____日历天

延期违约金：100 元/日历天
（从竣工日次日起算至实际交付日）

以下情况不计算延期（但需乙方书面通知+甲方确认）：
1. 甲方书面确认的增项导致的延期（增项对应的天数）
2. 不可抗力（如当地政府发布的停水停电超3日、自然灾害）
3. 以下情况不能作为延期的理由（看清楚！）：
   - 材料断货（这是乙方组织不力）
   - 施工人员调配不开（这是乙方管理问题）
   - 天气（雨天/潮湿天——这是乙方应预判的）
   - 甲方未配合验收（除非甲方确实3日内无法到场）
```

### 3.4 付款条款

| 节点 | 支付比例 | 金额 | 支付条件 |
|------|------|------|------|
| 签约 | **20%** | ___元 | 合同双方签字盖章后 |
| 水电验收 | **20%** | ___元 | 水电验收合格，双方签字确认 |
| 泥木验收 | **20%** | ___元 | 防水闭水试验通过、瓷砖/木工验收合格，双方签字 |
| 油漆验收 | **20%** | ___元 | 油漆/墙面验收合格，双方签字 |
| 竣工验收 | **15%** | ___元 | 全部专业验收通过，双方签字 |
| 居住3个月后 | **5%** | ___元 | 入驻满3个月，复验无问题后支付 |

**关键原则：**
- 不接受签约付40%/50%/60%
- 不验收不付款，不合格不签字
- 最后5%是你的终极保障
- 每次付款前必须拿到业主签字的验收单

### 3.5 增项管理条款

合同必须写明增项管理流程：

```
1. 任何增项（包括乙方建议或甲方提出）必须填《工程变更单》
2. 《工程变更单》必须写明：变更内容、材料品牌型号、数量、单价、总价
3. 甲方签字确认后方可施工
4. 未经甲方书面签字的增项，甲方不承担任何费用
5. 口头承诺或微信/电话约定的增项，甲方不承认其效力
6. 单个增项金额超过 500 元或累计增项超过 2000 元时，
   双方须签署补充协议
```

### 3.6 质保条款

```
质保期从竣工验收合格之日起算：

- 水电隐蔽工程：5年
- 防水工程：5年
- 表面工程（墙面/天花/瓷砖/地板）：2年
- 全屋定制（柜体/门板/五金）：2年
- 室内门/门窗套：2年
- 卫浴洁具：按品牌厂家售后标准

质保期内出现质量问题：
1. 甲方以书面方式（含微信+拍照）通知乙方
2. 乙方在7个工作日内安排人员上门查看并免费维修
3. 同一质量问题连续维修2次仍未解决的，甲方有权要求更换
4. 乙方逾期不回应或无法联系的，甲方自行安排维修，
   费用乙方据实报销 + 甲方可从未付尾款中直接扣抵
```

### 3.7 其他必须写入合同的条款

1. **安全责任：** 施工期间现场发生的一切安全事故（含人身伤亡、财产损失），责任由乙方承担。
2. **成品保护：** 每阶段施工完成后，乙方负责对已完成部分进行成品保护。若后续施工损坏已完工部分，乙方免费修复。
3. **全屋保洁：** 竣工验收移交前，乙方负责全屋开荒保洁一次（含窗户内外）。
4. **垃圾清运：** 施工期间产生的所有建筑垃圾，由乙方负责运输至小区指定堆放点（如小区无指定点，乙方负责外运合法消纳）。
5. **不得转包：** 本合同项下全部工程不得转包给第三方。如发现转包，甲方有权立即解除合同，乙方退还全部已付款 + 支付30%违约金。
6. **设计师到场：** 设计师必须在每个验收节点到场（水电/泥木/油漆/竣工）+ 全屋定制安装到场不少于2次。
7. **监理权：** 甲方有权聘请第三方监理，乙方应予以配合。
8. **发票：** 乙方应在收到每笔款项后5个工作日内开具等额发票。

## 第四阶段：主体拆改（5-7天，最容易违规）

### 4.1 开工前的保护

- 入户门：包门套+保护膜
- 窗户：保护膜贴里面
- 燃气表：用保护罩包好（弄坏了燃气公司让你整栋楼停气）
- 下水口：全部用盖子堵好，防止建筑垃圾掉落堵塞
- 公共区域（走廊到电梯）：铺地面保护垫

### 4.2 拆墙避坑

| 坑 | 怎么避 |
|------|------|
| 动承重墙 | 对照图纸确认。承重墙用手敲是"实"的，非承重墙声音发空。动了就是安全问题，被举报要恢复原状+罚款 |
| 拆墙太暴力 | 先用切割机切出边界，再用电锤。直接砸=砸裂相邻墙体+楼下天花板 |
| 铲墙皮不彻底 | 铲到水泥砂浆层才算完，残留白色腻子=不合格 |
| 建筑垃圾乱堆放 | 装修公司负责运到小区指定点，每天清理 |

### 4.3 砌墙避坑

| 坑 | 怎么避 |
|------|------|
| 新墙不植筋 | 每50cm高度植入一根钢筋连接新旧墙，不植筋=后期开裂 |
| 不挂钢丝网 | 新旧墙交接处挂钢丝网，宽15-20cm，防开裂 |
| 卫生间新墙不做地梁 | 卫生间砌墙前必须先倒地梁（混凝土导墙），高20cm，防潮水上墙 |
| 门洞不加过梁 | 所有门洞上方必须加水泥过梁（钢筋混凝土/槽钢），不能直接用砖搭，用角钢替代也可 |
| 顶砖斜砌 | 墙顶与梁/楼板之间的缝隙用砖斜砌+砂浆填满，不能直砌留缝 |
| 墙砌歪了 | 每砌完一层用水平尺量，误差超过3mm=拆了重砌 |

### 4.4 拆改验收

| 验收项 | 标准 | 工具 |
|------|------|------|
| 拆墙位置 | 与图纸一致 | 卷尺 |
| 新砌墙垂直度 | ≤3mm | 2米靠尺+塞尺 |
| 新砌墙平整度 | ≤3mm | 2米靠尺+塞尺 |
| 植筋 | 已做（每50cm） | 施工时看/要求拍照 |
| 地梁 | 卫生间墙已做 | 施工时看/要求拍照 |
| 钢丝网 | 新旧墙交接处已挂 | 抹灰前看 |
| 铲墙皮 | 到水泥层 | 目测 |
| 门洞过梁 | 已加 | 目测 |

**签字条件：** 以上全部项目验收合格 → 拆改验收签字。

## 第五阶段：水电改造（7-10天，最重要！）

### 5.1 开工前你必须准备的

**提前做好插座/开关/用水点清单。** 带着清单去现场一个一个点位对。

#### 各区域插座配置建议（131㎡）

**玄关（2-3个）**
- 鞋柜中间：1个五孔（烘鞋器/充电）
- 弱电箱旁：1个五孔（供猫/路由器）
- 离地30cm：1个备用

**客厅（12-15个）**
- 电视墙下方：5-6个五孔 + 1个网线 + 1个电视线
- 沙发两侧（离地30cm）：各2个五孔（带USB）
- 电视墙预埋50管（走线用）
- 阳台：1个五孔（扫地机器人）
- 窗帘盒旁：1个五孔（电动窗帘预留）

**餐厅（4-5个）**
- 餐桌旁：1个地插（火锅用）或墙插离地30cm
- 餐边柜：3个五孔（饮水机/咖啡机/充电）

**厨房（10-14个，全部带开关）**
- 台面上方（离地110-120cm）：6个带开关五孔
  - 电饭煲、微波炉、烧水壶、空气炸锅、豆浆机、预留
- 油烟机：1个（隐蔽在吊柜内）
- 冰箱：1个（离地30cm，**单独回路**）
- 燃气报警器：1个（离地30cm，靠近燃气表）
- 净水器：1个（水槽下方）
- 垃圾处理器：1个（水槽下方）
- 洗碗机：1个（水槽下方或旁边）
- 燃气热水器：1个

**主卧（8-12个）**
- 床头两侧（离地70-80cm）：各2个五孔（带USB）+ 双控开关
- 床尾/电视墙：2-3个五孔
- 梳妆台/书桌：2-3个五孔 + 网线
- 窗帘盒：1个（电动窗帘）
- 空调：1个16A插座（单独回路）

**次卧×2（各6-8个）**
- 同主卧逻辑，缩小

**书房/客卧（5-8个）**
- 书桌：3个五孔 + 网线
- 其余同卧室

**卫生间×2（各6-8个）**
- 洗手台旁：1个五孔（吹风机/电动牙刷充电）
- 马桶旁：1个五孔 + 防水盒（智能马桶盖预留）
- 洗衣机：1个（如放卫生间）
- 热水器：1个16A（如用电热水器）
- 电热毛巾架：1个（预留）
- 镜前灯：1根线

**生活阳台（3-5个）**
- 洗衣机：1个
- 烘干机：1个
- 热水器：1个（如放阳台）

**全屋合计约70-90个插座。** 套餐装修公司一般只含50个。超出多少 + 单价多少，必须写入合同。

### 5.2 水电材料验收

材料进场时你必须到。拿合同附件逐项对：
- 电线的品牌、平方数（看线身喷码）
- 水管品牌、壁厚（看管壁印刷）
- 线管品牌、颜色（新管不是回收料）
- 网线看线标（CAT5E/CAT6）
- 强弱电箱的品牌、回路数

### 5.3 施工中的细节

**电路：**
1. 线管横平竖直，大弯工艺（活线，后期能抽动）
2. 一根线管内不超过3根线（2.5mm²红线穿2根）
3. 强弱电管间距≥30cm，交叉处包锡纸
4. 管卡间距≤80cm（管卡=固定线管的卡子）
5. 插座底盒必须上保护盖（不然水泥会堵死）
6. 空调、厨卫用4mm²线，普通插座2.5mm²，照明1.5mm²
7. 开槽：承重墙横槽≤30cm、非承重墙≤50cm

**水路：**
1. 走顶不走地（走顶漏水一眼看见，走地等楼下找上来）
2. 左热右冷，间距15cm
3. 冷热水管全部用热水管（壁厚3.4mm以上），品牌统一
4. 排烟管、下水管必须用隔音棉包好再封，不然楼上冲水声很吵
5. 排水管坡度≥1%

**卫生间等电位：** 很多装修公司偷懒不接！打开等电位端子箱看有没有接线。等电位可以防止触电，是强规。

### 5.4 水电验收

**水管打压测试（你必须亲眼看着）：**
- 打到0.8MPa（8公斤压力）
- 保持30分钟
- 掉压≤0.05MPa为合格
- 各接头用纸巾擦拭，不能有渗水

**电路测试：**
- 验电器测两个插座：左零右火上接地
- 各回路开关测试：对应断路/对应照明
- 漏电保护器按测试按钮，必须跳闸
- 网线用测线仪测每一路

**自己拍照留档：**
- 每面墙都拍
- 卷尺入镜 (离开地面/墙角各多少厘米)
- 每个空间的四面墙各一张照片
- 不靠装修公司的"竣工图"，自己拍的最靠谱
- 以后打孔、挂画、装挂钩全靠这份照片

### 5.5 签字条件

- 水管打压通过（有压力表拍照）
- 电路全部通电测试通过
- 排水接好、倒水通畅
- 所有点位数量和位置与图纸一致
- 自己拍照留档已完成

**有任何一项没通过 = 不签字 = 不付款 = 不进入下一阶段**

## 第六阶段：泥瓦工（15-20天，最看手艺的阶段）

### 6.1 防水（最不能省）

**施工顺序：**
1. 清理基面 → 所有灰尘/松动颗粒清掉
2. 管根、阴角堵漏王封堵 + 圆弧处理
3. 第一遍防水涂料（纵横交叉刷）
4. 干透后（至少6小时）→ 第二遍防水涂料
5. 干透后（至少24小时）→ 闭水试验

**重点部位：**
- 管根（下水管根部）：堵漏王 + 防水附加层——漏水50%都在这儿
- 阴角（墙地交接处）：圆弧R角处理 + 防水附加层
- 门槛石：安装后两边都要刷堵漏王——水从这儿渗出去
- 卫生间门洞侧面：墙的两侧延展刷防水各30cm——防止返潮

**防水高度：**
- 淋浴区：≥1.8m（必须用柔韧型防水）
- 洗手台区：≥1.2m
- 其余墙面：≥30cm
- 厨房：地面 + 墙面30cm
- 阳台（如有洗衣机）：地面 + 墙面防水

### 6.2 闭水试验

1. 堵住所有地漏（用堵水球/沙袋）
2. 放水，水深没过地面3-5cm
3. 至少**48小时**
4. 联系楼下邻居，打开他们的卫生间天花板，用手摸、用纸巾擦拭，确认无任何渗水
5. 到楼下看的过程要拍照。有渗水痕迹 = 返工，直到不漏为止。

### 6.3 贴砖

**进场验砖：**
- 品牌、型号、色号与合同是否一致
- 批次号（不同批次可能有色差）
- 开3-5片看有没有崩角、色差、平整度问题

**贴砖顺序：** 先贴墙砖 → 再贴地砖（墙压地，排水更好，也更美观）

**注意：**
1. 全瓷砖（吸水率<0.5%）必须刷背胶 + 用瓷砖胶，纯水泥贴不住
2. 瓷片（吸水率高）泡水2小时以上再贴，不然吸水泥水分空鼓
3. 砖缝里的十字卡要清理干净，方便后期做美缝
4. 卫生间地砖做成坡向地漏（坡度≥1%），铺后用泼水来测排水
5. 厨房烟道用红砖包管，不能直接贴砖（烟道冷热变化大，贴砖会拉裂）
6. 所有开孔（底盒/出水口）必须用开孔器开圆孔，不能砸豁口。

**阳角处理：**
- 海棠角（45度切口拼直角）：最好看，对师傅手艺要求高
- 阳角条（金属/PVC）：施工简单，保护性好，但颜值略低
- 直接对碰：会露出毛边，一般不这么做

### 6.4 泥瓦验收

**防水验收：**
| 项 | 标准 | 检查方法 |
|------|------|------|
| 闭水试验 | 48小时楼下无渗水 | 去邻居家看 |
| 防水高度 | 淋浴≥1.8m | 卷尺量 |

**贴砖验收：**
| 项 | 标准 | 检查方法 |
|------|------|------|
| 空鼓率 | 单砖中心不允许空鼓，整屋空鼓率≤5% | 空鼓锤逐块敲 |
| 平整度 | ≤2mm/2m | 2米靠尺+塞尺 |
| 垂直度 | ≤2mm/2m | 2米靠尺 |
| 阴阳角方正 | ≤3mm | 直角尺 |
| 缝隙均匀 | 误差≤1mm | 目测十字缝+塞尺 |
| 地面坡度 | 地漏最低点，不积水 | 泼水+水平尺看坡向 |
| 地漏 | 回字形铺贴，低于砖面2-3mm | 目测 |
| 开孔 | 圆孔完整，无豁口 | 逐孔看 |
| 崩边崩瓷 | ≤0.5mm | 目测+手摸边角 |
| 墙地对缝 | 按设计图纸的对缝是否对齐 | 目测 |
| 表面 | 无划痕、无水泥污染、无色差 | 侧光看 |

### 6.5 美缝

贴完砖等水泥干透（至少**7天**，不能用赶工借口）再做。
做之前清缝：用吸尘器 + 刀片把缝里的水泥灰清干净。
颜色选跟瓷砖接近的最安全。

## 第七阶段：木工（5-8天）

**吊顶三要素：轻钢龙骨 + L型转角 + 防锈处理**

1. 龙骨全部用轻钢龙骨——不能是木龙骨，木龙骨吸潮变形 + 不防火
2. 副龙骨间距≤40cm，主龙骨间距≤1m
3. **转角必须是一块L型整板**，不能两块板拼接（一拼后期肯定开裂）
4. 石膏板接缝留3-5mmV型缝，方便后面嵌缝
5. 螺丝点防锈漆——每个螺丝！后期铁锈会透过乳胶漆
6. 窗帘盒预留足够深度（电动双帘≥20cm）
7. 如果需要装筒灯/射灯——在木工阶段就把开孔尺寸定好

**木工验收：**

| 验收项 | 标准 | 检查方法 |
|------|------|------|
| 龙骨材质 | 轻钢≠木 | 目测 |
| 龙骨间距 | 主≤1m、副≤40cm | 卷尺 |
| 转角 | L型整板 | 目测每个转角 |
| 接缝 | V型3-5mm | 目测+尺 |
| 螺丝 | 无穿纸面 | 手摸+目测 |
| 水平度 | ≤3mm/2m | 激光水平仪/靠尺 |
| 预留孔 | 灯/出风口尺寸对 | 卷尺对照图纸 |

## 第八阶段：油漆（12-18天，跳过梅雨季！）

**恩施气候提醒：** 梅雨季（6月-7月）湿度85%+，腻子干不透、乳胶漆无法成膜、后期必然开裂起皮发霉。如果工期撞上这个月——**暂停**。

### 8.1 施工步骤

```
墙固 → 嵌缝石膏(石膏板缝/线槽/新旧墙交接)
     → 第一遍腻子(干透)
     → 第二遍腻子(干透)
     → 第三遍腻子(干透后进行打磨)
     → 底漆(一遍)
     → 面漆第一遍(干透)
     → 面漆第二遍
```

每一步必须干透才能进入下一步。催进度 = 后续开裂。

### 8.2 注意事项

1. 阴阳角必须用阴阳角条找直
2. 所有螺丝先点防锈漆再刮腻子
3. 石膏板接缝处贴牛皮纸/防裂网
4. 线槽处、新旧墙交接处局部挂网
5. 乳胶漆到货必须当面扫防伪码验证
6. 调色漆必须一次性调好（分次调会有色差）+ 留备用漆

### 8.3 油漆验收

| 验收项 | 标准 | 检查方法 |
|------|------|------|
| 腻子层 | 无裂纹、无沙眼 | 目测 |
| 打磨 | 平整光滑无波浪 | 强光手电筒从侧面打光看整面墙 |
| 阴阳角 | 方正、垂直 | 直角尺 |
| 底漆 | 已刷 | 看颜色（刷过底漆墙面微微发暗） |
| 面漆 | 无色差、无流坠、无起泡、无刷痕 | 白天自然光 + 晚上手电筒侧照 |
| 分色 | 边界平直，不相互污染 | 看天花板与墙的交界处 |

## 第十阶段：全屋定制安装（5-7天，全程人盯人）

**这是整个装修最容易出错的环节，你必须全程在场。**

### 10.1 验货（安装师傅开包、你挨个点）

| 验收项 | 标准 |
|------|------|
| 柜体板 | 每块板背面有品牌LOGO。厚度18mm（拿卡尺量）≠ 16mm |
| 柜门 | 颜色与你在店里挑的一致。每扇门打开看正反面 |
| 封边 | 平滑、无崩边、无溢胶。尤其看四个角 |
| 五金 | 铰链/拉手/滑轨的品牌与合同一致 |
| 背板 | 厚度与合同一致（合同写的9mm就现场量，不能是5mm） |

### 10.2 安装中

1. 安装师傅打孔前必须对照水电照片——打穿水管/电线管就废了
2. 柜体靠墙侧必须用水平仪校准——柜子歪了门调不平
3. 柜体与墙面交接处用防霉玻璃胶密封——卫生间必须防霉型
4. 遇到需要切割开的板子，看截面有没有绿色防伪码（ENF级）

### 10.3 全屋定制验收

| 验收项 | 标准 | 检查方法 |
|------|------|------|
| 板材材质/厚度 | 与合同一致 | 卡尺量、看板身LOGO、看截面颜色/防伪 |
| 气味 | 正常木材味，不能刺鼻 | 闻——刺鼻=环保不达标 |
| 封边 | 无脱落/崩边/溢胶/色差 | 手摸每条边 |
| 柜体垂直度 | ≤2mm | 靠尺/水平仪 |
| 柜体水平度 | ≤2mm | 水平尺 |
| 门板平整度 | 无弯曲变形 | 侧光看 + 手摸 |
| 门缝均匀度 | 所有门缝一致，误差≤1mm | 目测 + 塞尺 |
| 门板闭合 | 无碰撞、无摩擦、关门有缓冲 | 每扇门开关3次 |
| **抽屉** | 抽拉顺畅、满载不卡、有缓冲 | 每个装几本书试 |
| **铰链** | 品牌正确、阻尼正常、无生锈 | 查LOGO+手试 |
| 拉手 | 牢固、水平、位置统一 | 手拉+水平尺 |
| 反弹器 | 按弹出、推回卡住 | 试3次 |
| 拉直器 | 超1.5m门板已装 | 目测 |
| 挂衣杆 | 牢固 | 双手下拉 |
| 柜内灯带 | 亮、手扫/开门感应灵敏 | 实际测试 |
| 打胶 | 柜体与墙面接触处全部密封 | 目测+手摸 |
| 台面 | 水平、接缝<0.5mm、打磨光滑、挡水条 | 手摸接缝 + 看反射 |

**有问题的当场写整改清单，双方签字、约好整改完成日期。不整改完成，不付下一笔款。**
