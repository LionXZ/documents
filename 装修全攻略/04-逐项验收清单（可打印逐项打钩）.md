## 0. 开工前交底（正式施工前）

这是最容易忽略的环节。开工前你必须到场，和设计师、工长三人一起确认。

| # | 交底内容 | ✅/❌ | 备注 |
|---|------|------|------|
| 1 | 对照图纸确认每面墙的拆改位置 | | |
| 2 | 弹线定位：用墨线弹出所有需要拆/砌的墙体位置 | | |
| 3 | 水平基准线弹出（全屋统一1米线） | | |
| 4 | 水电点位逐一确认（开关/插座/出水/排水位置） | | |
| 5 | 确认入户门/窗户/燃气表保护是否做好 | | |
