**颗粒板 vs 多层板 vs 生态板：**
- 颗粒板（ENF/MDI胶）：稳定性最好，做柜门不变形，性价比最高。很多人被忽悠"颗粒板=差"，但爱格也是颗粒板。
- 多层实木：防潮好于颗粒板，厨房/阳台柜可选。但做柜门易变形（超1.5m必加拉直器）。
- 生态板（大芯板）：木工现场打柜子用的多，定制工厂不用，环保取决于胶。

## 三、如何验证材料真伪

### 3.1 乳胶漆验真

立邦漆假货占比极高，必须掌握：

1. **看桶身** — 桶身印刷清晰，桶底有生产日期喷码
2. **扫防伪码** — 每桶都有唯一防伪码，用"立邦"官方微信/APP扫描
3. **官网验证** — 手动输入防伪码到品牌官网查询
4. **电话验证** — 拨打400防伪电话查询
5. **看封口** — 原厂封口完整、密封膜平整。开封过的不能要
6. **留空桶** — 全部用完的空桶留着，别让师傅带走（防止调包）

### 3.2 电线验真

1. 看线身喷码：品牌 + 规格（如"熊猫 BV 2.5mm²"）+ 执行标准
2. 拉绝缘层：拔不脱（包紧铜芯）
3. 看截面：铜芯紫红色，不发黑不氧化
4. 用千分尺量铜芯直径：
   - 1.5mm² = 直径1.38mm
   - 2.5mm² = 直径1.78mm
   - 4mm² = 直径2.25mm

### 3.3 水管验真

1. 看管身喷码：品牌 + 规格 + 执行标准 + 生产日期
2. PPR管燃烧有蜡烛味（不是刺鼻塑料味），熔接后冷却成为一个整体
3. 壁厚：管身印有S3.2/S4/S5系列。S3.2是热水管（壁厚≥3.4mm）

### 3.4 板材验真（全屋定制）

1. 看板身侧面或背面的**品牌防伪LOGO**
2. 万华禾香板：截面有绿色纤维颗粒 + 刮板面有淡淡秸秆清香（非刺鼻味）
3. 兔宝宝/莫干山：板面有防伪水印或钢印
4. 爱格板：每个花色有编号，看封边条上的花色编号与板上是否一致

### 3.5 五金验真

1. 百隆铰链：上面有Blum标志+奥地利制造钢印
2. 海蒂诗：LOGO清晰、表面电镀均匀
3. 东泰DTC：LOGO在铰臂，字体清晰

## 四、用量估算速查

### 4.1 瓷砖用量

```
瓷砖数量 = (铺贴面积 ÷ 单砖面积) × 1.05（5%损耗）
```

| 瓷砖规格 | 单砖面积 | 1㎡用量 | 100㎡(含损耗) |
|------|------|------|------|
| 300×300 | 0.09㎡ | 12片 | 1167片 |
| 300×600 | 0.18㎡ | 6片 | 584片 |
| 400×800 | 0.32㎡ | 4片 | 329片 |
| 600×600 | 0.36㎡ | 3片 | 292片 |
| 800×800 | 0.64㎡ | 2片 | 165片 |
| 600×1200 | 0.72㎡ | 2片 | 146片 |
| 750×1500 | 1.125㎡ | 1片 | 94片 |

> 全屋瓷砖面积≈套内面积 × (1.2~1.4)，即105㎡套内≈130-150㎡铺贴面积（含墙面）

### 4.2 乳胶漆用量

```
面漆用量(L) = 涂刷面积(㎡) × 遍数 ÷ 理论涂布率(㎡/L)
1桶5L面漆可涂 ≈ 30-35㎡ (一遍)
```

| 涂刷面积 | 2遍面漆用量 | 底漆用量 |
|------|------|------|
| 100㎡ | 6-7L（2桶5L） | 3-4L（1桶5L） |
| 200㎡ | 12-14L（3桶5L） | 6-8L（2桶5L） |
| 300㎡ | 18-20L（4桶5L） | 9-10L（2桶5L） |

> 涂刷面积 ≈ 套内面积 × 2.5~3（含天花）

### 4.3 瓷砖胶/背胶用量

| 砖的尺寸 | 厚度 | 墙面材料 |
|------|------|------|
| ≤300×600 | 水泥砂浆即可 | 砖墙——OK |
| ≤400×800 | 水泥砂浆 + 背胶 | 砖墙——OK |
| >400×800 / 全瓷砖 | 瓷砖胶C2 + 背胶 | 砖墙/混凝土墙 |
| 600×1200 / 750×1500 | 瓷砖胶C2 + 背胶 | 任何墙 |

### 4.4 美缝用量

```
美缝剂支数 = 缝总长(m) × 缝深(mm) × 缝宽(mm) ÷ 单支容量(mL)
单支400mL美缝剂 ≈ 约8-10㎡(800×800砖，2.5mm缝)
```
