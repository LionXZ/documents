## 快速进入

### 如果你现在是选公司阶段
1. 读 `01` → 逐项了解每个大项谈到什么程度才算谈完
2. 读 `02` → 第一到第三阶段（选公司/谈价/签合同）
3. 签合同时拿着 `03` 让对方逐项填写
4. 量房时对照 `05` 的尺寸表

### 如果你已经签了合同马上开工
1. 读 `02` → 第四阶段起（拆改/水电/泥瓦…）
2. 每阶段验收时打印 `04` 对应表格
3. 验材料真伪对照 `05` 第三章

### 如果你已经做到某个阶段了
直接跳到 `02` 和 `03` 的对应章节即可。

## 核心原则（永远记住）

1. **不验收不付款** — 不合格坚决不签字，不付下一笔款
2. **口头承诺=零** — 所有东西写进合同附件
3. **你不在场不施工** — 水电定位、水电验收、闭水试验、贴砖验收、全屋定制安装，你必须在
4. **过程留痕** — 每一步拍照+微信记录，纠纷时这是你的证据
5. **最后5%不急着付** — 入住3个月没问题再给
